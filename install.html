Instalação


Iremos dividir a instalação em duas partes, primeiro vamos montar um ambiente virtual e depois a instalação do Djando 1.10


Ambiente virtual

Então, vamos criar um ambiente virtual(também chamado um virtualenv). Isso isolará sua configuração Python/Django em uma base por projeto, significa que quaisquer mudanças que fizer em um website não afetará quaisquer outras aplicações que estiver desenvolvendo a parte.

Para este tutorial usaremos um novo diretório tutorial do seu diretório home:

mkdir tutorial
cd tutorial

Nós vamos fazer um virtualenv chamado myvenv . O formato geral desse comando é:


Linux: python3 -m venv myvenv

Windows: C:\Usuário\Nome\tutorial> C:\Python34\python -m venv myvenv


O comando que acabamos de executar criará um diretório chamado myvenv (ou seja o nome que você escolheu) que contém o nosso ambiente virtual, agora vamos ativar ela.


Linux: ~/tutorial$ source myvenv/bin/activate

Windows: C:\Usuário\Nome\tutorial> myvenv\Scripts\activate


Perceba que o prefixo (myvenv) aparece!


Instalando o Django

Agora que você tem a sua virtualenv iniciado, você pode instalar Django usando pip .
No console, execute pip install django==1.10

(myvenv) ~$ pip install django==1.10
Downloading/unpacking django==1.10
Installing collected packages: django
Successfully installed django
Cleaning up...

É isso! Agora você está (finalmente) pronto para criar uma aplicação Django!
